1. **下载文件**：请下载本仓库提供的“excel中161个VBA_自定义函数超级实用”文件。
2. **导入Excel**：打开Excel，按下`Alt + F11`进入VBA编辑器，选择“插入” -> “模块”，然后将下载的文件内容粘贴到模块中。
3. **使用函数**：在Excel中直接使用这些自定义函数，例如在单元格中输入`=MyFunction(参数)`即可调用相应的函数。
